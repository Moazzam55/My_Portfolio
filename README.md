# Moazzam Hayat Khan — Portfolio

My personal portfolio site — a single-page site built with plain HTML, CSS and JavaScript (no frameworks, no build step). It covers who I am, what I work on, and how to reach me.

## About

I'm a final-year CSE student and Data Analyst Intern, splitting my time between data pipelines and front-end builds. This site is where both sides live — the data/ML work from my internship and coursework, and the small HTML/CSS/JS projects I build on the side.

| Section | What's there |

## Built with

- **HTML5** — semantic structure
- **CSS3** — custom properties, Flexbox & Grid, no framework
- **JavaScript (vanilla)** — nav highlighting, scroll reveals, tab switching, typewriter effect, contact form handoff
- **Google Fonts** — Space Grotesk, Inter, JetBrains Mono

No build tools, no dependencies to install — it's just three files.

## Running it locally

Clone the repo and open the file — that's it:

```bash
git clone https://github.com/Moazzam55/portfolio.git
cd portfolio
open index.html      # or just double-click it
```

For live-reload while editing, any static server works, e.g.:

```bash
npx serve .
```

## Deploying to GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under **Source**, pick the `main` branch and `/ (root)` folder.
4. Save — the site publishes at `https://<username>.github.io/<repo-name>/`.
